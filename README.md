# mqsampler

MQ JMeter Extension.

A [JMeter](http://jmeter.apache.org/) Plugin to publish mq message on Topic via ALIBABA MQ SDK.).

## Install

Build the extension:

    mvn package

Install the extension `mqmeter-x.y.z.jar` into 

    `$JMETER_HOME/lib/ext`.
